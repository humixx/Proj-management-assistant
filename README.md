# Project Management Assistant

A comprehensive project management assistant with AI-powered chat, task management, document processing, and Slack integration.

## Project Structure

This project follows a monorepo structure with separate backend and frontend applications.

### Backend
- FastAPI-based REST API
- PostgreSQL with pgvector for vector storage
- AI agent with tool-based execution
- Document processing and RAG capabilities
- Slack integration

### Frontend
- Next.js with TypeScript
- React components
- Real-time updates via WebSocket
- Zustand for state management

## Getting Started

See individual README files in `backend/` and `frontend/` directories for setup instructions.
